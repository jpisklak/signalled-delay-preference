# README: signalled-delay-preference

## Overview

This repository contains the data and R code for the study the study titled "Waiting for Reward: Preference for Signalled Delays Increases with Average Wait Time" by Margaret A. McDevitt, Jeffrey M. Pisklak, Roger M. Dunn, and Marcia L. Spetch.

**Associated OSF Project:**

* Spetch, M. L., Dunn, R. M., Pisklak, J. M., & McDevitt, M. A. (2025, December 3). Does Preference for Signalled Delays Increase With Average Wait Time. [https://osf.io/zcwhp](https://osf.io/zcwhp)

**Associated OSF Pre-Registration:**

* Spetch, M. L., Dunn, R. M., Pisklak, J. M., & McDevitt, M. A. (2025, January 16). Does Preference for Signalled Delays Increase With Average Wait Time. [https://doi.org/10.17605/OSF.IO/9AS82](https://doi.org/10.17605/OSF.IO/9AS82)


## Contents

The directory, file, and variable names within this repository are designed to be self-explanatory.

* `data/choice_data.csv` is a comma-separated values (.CSV) spreadsheet comprising all the raw choice response data.

* `data/latency_data.csv` is a comma-separated values (.CSV) spreadsheet comprising all the raw latency data. The columns `p1_1`, `p1_2`, `p1_3`, and so on, refer to *phase 1: session 1*, *phase 1: session 2*, *phase 1: session 3*, etc.

* `data/harmonized_studies_supplement.csv` is a comma-separated values (.CSV) spreadsheet containing the results of studies noted in the supplemental portion of the manuscript.


Below are common abbreviations used throughout the code:

- `cp`: Choice Proportion
- `ch`: Choice
- `resp`: Responses
- `sh`: Short
- `lo`: Long
- `fun`: Function
- `il`: Initial Link
- `tl`: Terminal Link
- `sig`: Signaled
- `unsig`: Unsignalled
